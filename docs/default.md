# Default Layout Examples

Glove80 ships several stock layouts curated by MoErgo.  The `layouts/default` directory tracks the JSON artifacts
exported from the layout editor so they can be referenced alongside TailorKey and QuantumTouch.

## Canonical Layouts (accessed 8 Nov 2025)
- [Glove80 Factory Default Layout](https://my.glove80.com/#/layout/glove80)
- [Glove80 Factory Default Layout for macOS](https://my.glove80.com/#/layout/glove80-macos)
- [Mouse Emulation Example](https://my.glove80.com/#/layout/mouse-emulation)
- [Colemak Layout](https://my.glove80.com/#/layout/colemak)
- [Colemak-DH Layout](https://my.glove80.com/#/layout/colemak-dh)
- [Dvorak Layout](https://my.glove80.com/#/layout/dvorak)
- [Workman Layout](https://my.glove80.com/#/layout/workman)
- [Kinesis Advantage-like Layout](https://my.glove80.com/#/layout/kinesis)

These JSON files are stored verbatim; they are not regenerated from Python code yet.  Keeping them under version control
